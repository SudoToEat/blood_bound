# 鲜血盟约 (Blood Bound) 本地网页版

这是桌游《鲜血盟约》(Blood Bound) 的本地网页版实现，使用 React + TypeScript + Tailwind CSS 开发。

## 功能特点

- 中文界面
- 支持 6-12 名玩家
- 随机分配角色和阵营（吸血鬼或猎魔人）
- 每位玩家只能看到自己的身份
- 支持切换查看每位玩家的身份（用于线下把设备传给他看）
- **新功能：多设备访问** - 每个玩家可以通过手机扫码查看自己的身份
- 提供游戏规则说明
- 所有逻辑前端完成，无需后端支持

## 游戏简介

血契獵殺(Blood Bound)是一款陣營推理類桌遊。玩家会分成两大陣營(也有第三个陣營)，目标是找出并殺掉对方陣營的族长(等级数字最小的角色是族长)。在遊戲過程中，玩家的身份是秘密，只能透過攻擊彼此，來揭露對方的陣營和等級線索。線索出來後就要盡快推理找出目標，然後殺掉對手的族長。

## 安装与运行

### 前提条件

- Node.js (推荐 v14 或更高版本)
- npm 或 yarn

### 安装依赖

```bash
npm install
# 或
yarn
```

### 开发模式运行

```bash
npm run dev
# 或
yarn dev
```

### 构建生产版本

```bash
npm run build
# 或
yarn build
```

## 使用说明

### 单设备模式

1. 在首页选择玩家人数（6-12人）
2. 点击「下一步：创建房间」按钮
3. 点击「开始游戏」按钮（跳过房间创建步骤）
4. 在玩家列表中，点击「查看身份」按钮查看对应玩家的身份
5. 将设备传给对应的玩家，让他们查看自己的身份
6. 玩家查看完身份后点击「返回」按钮

### 多设备模式

1. 在首页选择玩家人数（6-12人）
2. 点击「下一步：创建房间」按钮
3. 点击「创建房间」按钮
4. 让每位玩家通过扫描二维码或使用链接访问自己的身份页面
5. 所有玩家准备好后，点击「开始游戏」按钮

游戏中的所有操作需要在线下进行，本应用仅用于身份分配

### 测试多设备功能

可以使用提供的测试脚本启动支持局域网访问的服务器：

```bash
chmod +x test_mobile_access.sh
./test_mobile_access.sh
```

或直接访问 `[游戏URL]?test=true` 进入测试页面
